# 🧠 Math Game - C++/CLI (Windows Forms)

Ushbu loyiha foydalanuvchining matematik hisoblash tezligini sinovdan o'tkazuvchi qiziqarli o'yin. Dastur Visual Studio muhitida **C++/CLI** texnologiyasi va **Windows Forms** interfeysi yordamida yaratilgan.

## 🚀 Dasturning imkoniyatlari
* **Tasodifiy misollar:** Dastur har safar har xil sonlar va matematik amallar (+, -, *, /, %) orqali misol hosil qiladi.
* **Vaqt nazorati:** O'yinchi 60 soniya ichida imkon qadar ko'p misol yechishi kerak.
* **Ballar tizimi:** Har bir to'g'ri javob uchun 1 ball beriladi.
* **Aqlli tekshiruv:** Bo'lish amallarida verguldan keyingi aniqlik 2 xonagacha hisobga olingan.
* **Xatoliklarni ushlash:** Noto'g'ri belgi yoki bo'sh maydon kiritilganda dastur ogohlantirish beradi.

## 🛠 Texnologiyalar
- **Til:** C++
- **Platforma:** .NET Framework (Windows Forms)
- **Muhit:** Visual Studio

## 🎮 O'yinni qanday o'ynash kerak?
1. **START** tugmasini bosing.
2. Ekranda paydo bo'lgan matematik misolni yeching.
3. Javobni matn maydoniga yozing va **NEXT** tugmasini bosing.
4. Vaqt tugagandan so'ng umumiy balingizni ko'rishingiz mumkin.
